1.1.5: Token: Constants (Numeric)

Problem Statement

Demonstrate the use of both **Integer Constants** (`100`) and **Floating-Point Constants** (`3.14159f`) by assigning and printing them.

// Lecture 1, Part 1, Sub-Problem 1.1.5 - Numeric Constants
#include <stdio.h>

int main() {
    const int MAX_VALUE = 100; // 100 is the Integer Constant
    const float PI = 3.14159f; // 3.14159f is the Floating-Point Constant

    printf("Max Value: %d\n", MAX_VALUE);
    printf("Pi: %.5f\n", PI);
    return 0;
}

1.1.6: Token: Constants (Char & String)

Problem Statement

Illustrate the difference between a **Character Constant** (single quotes) and a **String Literal** (double quotes).

// Lecture 1, Part 1, Sub-Problem 1.1.6 - Char and String Constants
#include <stdio.h>

int main() {
    char grade = 'A'; // 'A' is a Character Constant
    char course[] = "C Programming"; // "C Programming" is a String Literal

    printf("Grade: %c\n", grade);
    printf("Course: %s\n", course);
    return 0;
}

1.1.7: Token: Separators (Semicolon)

Problem Statement

Demonstrate the **semicolon (`;`)** as a crucial separator token used to terminate statements in C.

// Lecture 1, Part 1, Sub-Problem 1.1.7 - Semicolon Token
#include <stdio.h>

int main() {
    int x = 10; // Statement 1 ends with semicolon
    x = x + 5;  // Statement 2 ends with semicolon
    printf("Result: %d\n", x); // Statement 3 ends with semicolon
    return 0;   // Statement 4 ends with semicolon
}

1.1.11: Keyword: const (Defining Constant)

Problem Statement

Use the **keyword `const`** to create a read-only variable (`RATE`), confirming its value cannot be changed later in the program.

// Lecture 1, Part 1, Sub-Problem 1.1.11 - const Keyword
#include <stdio.h>

int main() {
    // 'const' keyword makes RATE read-only
    const float RATE = 0.05f; 
    
    float initial_amount = 1000.0f;
    float interest = initial_amount * RATE;

    printf("Rate: %.2f\n", RATE);
    printf("Interest: %.2f\n", interest);

    // RATE = 0.06f; // ERROR: Cannot modify a const variable
    return 0;
}

1.2.1: Practice: Read Two Integers and Print Separately

1.2.2: Practice: Calculate Quotient and Remainder

Scenario

Given two integers, `A` (dividend) and `B` (divisor), calculate and print the integer **quotient** and the **remainder**.

// Lecture 1, Part 2, Sub-Problem 1.2.2 - Quotient and Remainder
#include <stdio.h>

int main() {
    int A, B, quotient, remainder;
    
    scanf("%d %d", &A, &B); 

    // Calculate quotient using division operator (/)
    quotient = A / B;
    
    // Calculate remainder using modulus operator (%)
    remainder = A % B;

    printf("Quotient: %d\n", quotient);
    printf("Remainder: %d\n", remainder);

    return 0;
}

1.2.3: Practice: Swap Two Variables (Using Temporary)

Scenario

Accept two integers, `P` and `Q`. Swap their values using a temporary third variable, and print the new values of `P` and `Q`.

// Lecture 1, Part 2, Sub-Problem 1.2.3 - Variable Swap
#include <stdio.h>

int main() {
    int P, Q, temp;
    
    scanf("%d %d", &P, &Q); 
    
    // Swapping logic using a temporary variable
    temp = P; 
    P = Q;     
    Q = temp; 

    printf("New P: %d\n", P);
    printf("New Q: %d\n", Q);

    return 0;
}

1.2.4: Practice: Calculate Average of Three Integers

Scenario

Read three integers, calculate their sum, and then determine their average. The average must be printed as a floating-point number (at least one decimal place).

// Lecture 1, Part 2, Sub-Problem 1.2.4 - Calculate Average
#include <stdio.h>

int main() {
    int n1, n2, n3;
    float average;

    scanf("%d %d %d", &n1, &n2, &n3);
    
    // Sum the integers, cast one operand to float to force float division
    average = (float)(n1 + n2 + n3) / 3;

    // Print the average to one decimal place
    printf("Average: %.1f\n", average);

    return 0;
}
