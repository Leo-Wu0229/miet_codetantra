2.1.11: `printf()`: Printing Multiple Variables

    int i = 5;

2.1.13: Basic `scanf()`: Reading an Integer (`%d`)

Problem Statement

Use `scanf()` to read a single integer input from the user and print it back. Ensure the ampersand (`&`) is used for the variable.

// Lecture 2, Part 1, Sub-Problem 2.1.13 - Basic scanf int
#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    // Read an integer into the memory address of 'age'
    scanf("%d", &age); 
    printf("Your age is: %d\n", age);
    return 0;
}

2.1.19: `scanf()`: Suppressing Input (`%*d`)

Problem Statement

Read three integers, but ignore the second input using the suppression modifier (`*`) and only store and print the first and third integers.

// Lecture 2, Part 1, Sub-Problem 2.1.19 - Input Suppression
#include <stdio.h>

int main() {
    int first, third;
    
    printf("Enter three integers (e.g., 10 20 30):\n");
    // %*d reads the second integer but discards it

}

2.1.20: I/O for `double` (`%lf` in scanf, `%f` in printf)

Problem Statement

// Lecture 2, Part 1, Sub-Problem 2.1.20 - Double I/O
#include <stdio.h>

int main() {
    double large_num;
    
    printf("Enter a large decimal number:\n");

}

2.2.1: Practice: Reading Mixed Data Types

Scenario

Accept a character (initial), an integer (age), and a float (height in meters) in that order, and print a formatted summary sentence.

Example Input: `J 21 1.75`

// Lecture 2, Part 2, Sub-Problem 2.2.1 - Mixed Input
#include <stdio.h>

int main() {
    char initial;
    int age;
    float height;

    // Note the space before %c to handle whitespace
    scanf(" %c %d %f", &initial, &age, &height);

    return 0;
}

2.2.2: Practice: Simple BMI Calculation

Scenario

Read a person's **weight (kg, integer)** and **height (m, float)**. Calculate and print the BMI (BMI = weight / (height * height)), formatted to one decimal place.

Formula: $BMI = \frac{weight}{height^2}$

// Lecture 2, Part 2, Sub-Problem 2.2.2 - BMI Calculation
#include <stdio.h>

int main() {
    int weight; // integer kg
    float height; // float meters
    float bmi;

    scanf("%d %f", &weight, &height); 

    // Calculate BMI, ensuring float arithmetic
    bmi = weight / (height * height); 

    printf("BMI: %.1f\n", bmi);
    return 0;
}
